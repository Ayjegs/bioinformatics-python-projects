# DNA Sequence Analyzer
A beginner bioinformatics project written in Python.

## Description
This program analyzes a DNA sequence and reports:

- Sequence length
- Uppercase validation
- Base composition (A, T, G, C)
- Unique bases
- Start codon detection
- Reverse sequence
- Codon splitting
- DNA sequence validation
- GC content calculation

- Python 3
